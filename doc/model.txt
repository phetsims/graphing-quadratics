# Graphing Quadratics - model description

_Explore_ and _Standard Form_ screens use the standard form:

_y_ = _a_x<sup>2</sup> + _b_x + _c_

_Vertex Form_ screen uses the vertex form:

_y_ = _a_(_x_ - _h_)<sup>2</sub> + k

_Focus & Directrix_ screen uses an alternative vertex form, where _a_ is replaced with 1/(4_p_):

_y_ = (1/(4_p_))(_x_ - _h_)<sup>2</sub> + k
